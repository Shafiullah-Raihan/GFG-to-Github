<h2><a href="https://www.geeksforgeeks.org/problems/mother-vertex/1">Mother Vertex</a></h2><h3>Difficulty Level : Difficulty: Medium</h3><hr><div class="problems_problem_content__Xm_eO" style="--text-color: var(--problem-text-color);"><p style="font-family: Nunito, Bangla782, sans-serif;"><span style="font-size: 18px; font-family: Nunito, Bangla782, sans-serif;">Given a <strong style="font-family: &quot;Source Sans 3&quot;, Bangla782, sans-serif;">directed</strong> graph with <strong style="font-family: &quot;Source Sans 3&quot;, Bangla782, sans-serif;">V</strong> vertices labeled from 0 to V-1 and a list of edges <strong style="font-family: &quot;Source Sans 3&quot;, Bangla782, sans-serif;">edges[][]</strong>, where each edge is represented as [u, v] indicating a directed edge from vertex u to vertex v, find a <strong style="font-family: &quot;Source Sans 3&quot;, Bangla782, sans-serif;">Mother Vertex</strong> of the graph.</span></p>
<p style="font-family: Nunito, Bangla782, sans-serif;"><span style="font-size: 18px; font-family: Nunito, Bangla782, sans-serif;">A Mother Vertex is a vertex from which all other vertices can be reached.</span></p>
<ul style="font-family: Nunito, Bangla782, sans-serif;">
<li style="font-family: Nunito, Bangla782, sans-serif;"><span style="font-size: 18px; font-family: Nunito, Bangla782, sans-serif;">If multiple such vertices exist, return the one with the <strong style="font-family: &quot;Source Sans 3&quot;, Bangla782, sans-serif;">smallest</strong> value.</span></li>
<li style="font-family: Nunito, Bangla782, sans-serif;"><span style="font-size: 18px; font-family: Nunito, Bangla782, sans-serif;">If no such vertex exists, return <strong style="font-family: &quot;Source Sans 3&quot;, Bangla782, sans-serif;">-1</strong>.</span></li>
</ul>
<p style="font-family: Nunito, Bangla782, sans-serif;"><span style="font-size: 18px; font-family: Nunito, Bangla782, sans-serif;"><strong style="font-family: &quot;Source Sans 3&quot;, Bangla782, sans-serif;">Examples:</strong></span></p>
