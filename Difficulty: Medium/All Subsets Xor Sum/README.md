<h2><a href="https://www.geeksforgeeks.org/problems/sum-of-xor-of-all-possible-subsets/1">All Subsets Xor Sum</a></h2><h3>Difficulty Level : Difficulty: Medium</h3><hr><div class="problems_problem_content__Xm_eO"><p style="font-family: Nunito, Bangla662, sans-serif;"><span style="font-size: 18.6667px; font-family: Nunito, Bangla662, sans-serif;">Given an array <strong style="font-family: &quot;Source Sans 3&quot;, Bangla662, sans-serif;">arr[]</strong>, return the sum of the XOR of all elements for every possible <strong style="font-family: &quot;Source Sans 3&quot;, Bangla662, sans-serif;">subset </strong>of the array. </span><span style="font-size: 14pt; font-family: Nunito, Bangla662, sans-serif;">Subsets with the same elements should be counted multiple times.</span></p>
<p style="font-family: Nunito, Bangla662, sans-serif;"><span style="font-size: 14pt; font-family: Nunito, Bangla662, sans-serif;">An array <strong style="font-family: &quot;Source Sans 3&quot;, Bangla662, sans-serif;">a </strong>is a subset of an array <strong style="font-family: &quot;Source Sans 3&quot;, Bangla662, sans-serif;">b </strong>if a can be obtained from b by deleting some (possibly zero) elements of b.</span></p>
<p style="font-family: Nunito, Bangla662, sans-serif;"><span style="font-size: 14pt; font-family: Nunito, Bangla662, sans-serif;"><strong style="font-family: &quot;Source Sans 3&quot;, Bangla662, sans-serif;">Note:</strong> The answer is guaranteed to fit within a <strong style="font-family: &quot;Source Sans 3&quot;, Bangla662, sans-serif;">32-bit</strong> integer.</span></p>
<p style="font-family: Nunito, Bangla662, sans-serif;"><span style="font-size: 14pt; font-family: Nunito, Bangla662, sans-serif;"><strong style="font-family: &quot;Source Sans 3&quot;, Bangla662, sans-serif;">Examples:</strong></span></p>
<pre style="font-family: Nunito, Bangla662, sans-serif;"><span style="font-size: 14pt; font-family: Nunito, Bangla662, sans-serif;"><strong style="font-family: &quot;Source Sans 3&quot;, Bangla662, sans-serif;">Input</strong>: arr[] = [7, 2]</span><br style="font-family: Nunito, Bangla662, sans-serif;"><span style="font-size: 14pt; font-family: Nunito, Bangla662, sans-serif;"><strong style="font-family: &quot;Source Sans 3&quot;, Bangla662, sans-serif;">Output</strong>: 14</span><br style="font-family: Nunito, Bangla662, sans-serif;"><span style="font-size: 14pt; font-family: Nunito, Bangla662, sans-serif;"><strong style="font-family: &quot;Source Sans 3&quot;, Bangla662, sans-serif;">Explanation</strong>: Subsets are: [[], [7], [2], [7, 2]]</span><br style="font-family: Nunito, Bangla662, sans-serif;"><span style="font-size: 14pt; font-family: Nunito, Bangla662, sans-serif;">Sum of all XOR's = 7 + 2 + (7 ^ 2) = 14.</span></pre>
<pre style="font-family: Nunito, Bangla662, sans-serif;"><span style="font-size: 14pt; font-family: Nunito, Bangla662, sans-serif;"><strong style="font-family: &quot;Source Sans 3&quot;, Bangla662, sans-serif;">Input</strong>: arr[] = [1, 2, 3]</span><br style="font-family: Nunito, Bangla662, sans-serif;"><span style="font-size: 14pt; font-family: Nunito, Bangla662, sans-serif;"><strong style="font-family: &quot;Source Sans 3&quot;, Bangla662, sans-serif;">Output</strong>: 12</span><br style="font-family: Nunito, Bangla662, sans-serif;"><span style="font-size: 14pt; font-family: Nunito, Bangla662, sans-serif;"><strong style="font-family: &quot;Source Sans 3&quot;, Bangla662, sans-serif;">Explanation</strong>: Subsets are: [[], [1], [2], [3], [1, 2], [1, 3], [2, 3], [1, 2, 3]]</span><br style="font-family: Nunito, Bangla662, sans-serif;"><span style="font-size: 14pt; font-family: Nunito, Bangla662, sans-serif;">Sum of all XOR's = 1 + 2 + 3 + (1 ^ 2)&nbsp; + (1 ^ 3) + (2 ^ 3) + (1 ^ 2 ^ 3) = 12.</span></pre>
