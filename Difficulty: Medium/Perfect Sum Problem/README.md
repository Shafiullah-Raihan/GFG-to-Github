<h2><a href="https://www.geeksforgeeks.org/problems/perfect-sum-problem5633/1?page=2&sortBy=submissions">Perfect Sum Problem</a></h2><h3>Difficulty Level : Difficulty: Medium</h3><hr><div class="problems_problem_content__Xm_eO"><p style="font-family: Nunito, Bangla899, sans-serif;"><span style="font-size: 14pt; font-family: Nunito, Bangla899, sans-serif;">Given an array <strong style="font-family: &quot;Source Sans 3&quot;, Bangla899, sans-serif;"><code style="font-family: Nunito, Bangla899, sans-serif;">arr</code></strong> of non-negative integers and an integer <strong style="font-family: &quot;Source Sans 3&quot;, Bangla899, sans-serif;"><code style="font-family: Nunito, Bangla899, sans-serif;">target</code></strong>, the task is to count all subsets of the array whose sum is equal to the given <code style="font-family: Nunito, Bangla899, sans-serif;">target</code>.</span></p>
<p style="font-family: Nunito, Bangla899, sans-serif;"><span style="font-size: 14pt; font-family: Nunito, Bangla899, sans-serif;"><strong style="font-family: &quot;Source Sans 3&quot;, Bangla899, sans-serif;">Examples:</strong></span></p>
<pre style="font-family: Nunito, Bangla899, sans-serif;"><span style="font-size: 14pt; font-family: Nunito, Bangla899, sans-serif;"><strong style="font-family: &quot;Source Sans 3&quot;, Bangla899, sans-serif;">Input</strong>: arr[] = [5, 2, 3, 10, 6, 8], target = 10
<strong style="font-family: &quot;Source Sans 3&quot;, Bangla899, sans-serif;">Output: </strong>3
<strong style="font-family: &quot;Source Sans 3&quot;, Bangla899, sans-serif;">Explanation</strong>: The subsets {5, 2, 3}, {2, 8}, and {10} sum up to the target 10.</span></pre>
<pre style="font-family: Nunito, Bangla899, sans-serif;"><span style="font-size: 14pt; font-family: Nunito, Bangla899, sans-serif;"><strong style="font-family: &quot;Source Sans 3&quot;, Bangla899, sans-serif;">Input</strong>: arr[] = [2, 5, 1, 4, 3], target = 10
<strong style="font-family: &quot;Source Sans 3&quot;, Bangla899, sans-serif;">Output:</strong> 3
<strong style="font-family: &quot;Source Sans 3&quot;, Bangla899, sans-serif;">Explanation</strong>: The subsets {2, 1, 4, 3}, {5, 1, 4}, and {2, 5, 3} sum up to the target 10.</span></pre>
<pre style="font-family: Nunito, Bangla899, sans-serif;"><span style="font-size: 14pt; font-family: Nunito, Bangla899, sans-serif;"><strong style="font-family: &quot;Source Sans 3&quot;, Bangla899, sans-serif;">Input</strong><span style="font-family: Nunito, Bangla899, sans-serif;">: </span>arr[] = [5, 7, 8], target = 3<span style="font-family: Nunito, Bangla899, sans-serif;"><br style="font-family: Nunito, Bangla899, sans-serif;"></span><strong style="font-family: &quot;Source Sans 3&quot;, Bangla899, sans-serif;">Output:</strong> 0
<strong style="font-family: &quot;Source Sans 3&quot;, Bangla899, sans-serif;">Explanation</strong>: There are no subsets of the array that sum up to the target 3.</span></pre>
<pre style="font-family: Nunito, Bangla899, sans-serif;"><span style="font-size: 14pt; font-family: Nunito, Bangla899, sans-serif;"><strong style="font-family: &quot;Source Sans 3&quot;, Bangla899, sans-serif;">Input</strong><span style="font-family: Nunito, Bangla899, sans-serif;">: </span>arr[] = [35, 2, 8, 22], target = 0<span style="font-family: Nunito, Bangla899, sans-serif;"><br style="font-family: Nunito, Bangla899, sans-serif;"></span><strong style="font-family: &quot;Source Sans 3&quot;, Bangla899, sans-serif;">Output:</strong> 1
<strong style="font-family: &quot;Source Sans 3&quot;, Bangla899, sans-serif;">Explanation</strong>: The empty subset is the only subset with a sum of 0.</span></pre>
