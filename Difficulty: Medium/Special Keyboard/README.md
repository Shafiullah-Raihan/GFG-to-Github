<ul style="font-family: Nunito, Bangla814, sans-serif;">
<li style="font-family: Nunito, Bangla814, sans-serif;"><span style="font-size: 18px; font-family: Nunito, Bangla814, sans-serif;"><span style="font-size: 18px; font-family: Nunito, Bangla814, sans-serif;"><strong style="font-family: &quot;Source Sans 3&quot;, Bangla814, sans-serif;">Key 1:</strong> Prints a single character 'A' on the screen.</span></span></li>
<li style="font-family: Nunito, Bangla814, sans-serif;"><span style="font-size: 18px; font-family: Nunito, Bangla814, sans-serif;"><span style="font-size: 18px; font-family: Nunito, Bangla814, sans-serif;"><strong style="font-family: &quot;Source Sans 3&quot;, Bangla814, sans-serif;">Key 2 (Ctrl + A):</strong> Selects all the characters currently present on the screen.</span></span></li>
<li style="font-family: Nunito, Bangla814, sans-serif;"><span style="font-size: 18px; font-family: Nunito, Bangla814, sans-serif;"><span style="font-size: 18px; font-family: Nunito, Bangla814, sans-serif;"><strong style="font-family: &quot;Source Sans 3&quot;, Bangla814, sans-serif;">Key 3 (Ctrl + C):</strong> Copies the selected characters to a buffer.</span></span></li>
<li style="font-family: Nunito, Bangla814, sans-serif;"><span style="font-size: 18px; font-family: Nunito, Bangla814, sans-serif;"><span style="font-size: 18px; font-family: Nunito, Bangla814, sans-serif;"><strong style="font-family: &quot;Source Sans 3&quot;, Bangla814, sans-serif;">Key 4 (Ctrl + V): </strong>Pastes the content of the buffer onto the screen, appending it to the existing text.</span></span></li>
</ul>
<p style="font-family: Nunito, Bangla814, sans-serif;"><span style="font-size: 18px; font-family: Nunito, Bangla814, sans-serif;"><span style="font-size: 18px; font-family: Nunito, Bangla814, sans-serif;">Initially, the screen is empty and the buffer is also empty.</span></span></p>
<p style="font-family: Nunito, Bangla814, sans-serif;"><span style="font-size: 18px; font-family: Nunito, Bangla814, sans-serif;">Determine the <strong style="font-family: &quot;Source Sans 3&quot;, Bangla814, sans-serif;">maximum </strong>number of 'A' characters that can be displayed on the screen after performing<strong style="font-family: &quot;Source Sans 3&quot;, Bangla814, sans-serif;"> exactly n</strong> key presses.</span></p>
<p style="font-family: Nunito, Bangla814, sans-serif;"><strong style="font-family: &quot;Source Sans 3&quot;, Bangla814, sans-serif;"><span style="font-size: 18px; font-family: Nunito, Bangla814, sans-serif;">Examples :</span></strong></p>
<pre style="font-family: Nunito, Bangla814, sans-serif;"><span style="font-size: 18px; font-family: Nunito, Bangla814, sans-serif;"><strong style="font-family: &quot;Source Sans 3&quot;, Bangla814, sans-serif;">Input:</strong> n = 3
<strong style="font-family: &quot;Source Sans 3&quot;, Bangla814, sans-serif;">Output:</strong> 3
<strong style="font-family: &quot;Source Sans 3&quot;, Bangla814, sans-serif;">Explanation:</strong> With only 3 key presses, the best option is to press Key 1 each time. So, the screen shows "AAA" and the total number of A’s is 3.</span></pre>
<pre style="font-family: Nunito, Bangla814, sans-serif;"><span style="font-size: 18px; font-family: Nunito, Bangla814, sans-serif;"><strong style="font-family: &quot;Source Sans 3&quot;, Bangla814, sans-serif;">Input:</strong> n = 7
<strong style="font-family: &quot;Source Sans 3&quot;, Bangla814, sans-serif;">Output:</strong> 9
<strong style="font-family: &quot;Source Sans 3&quot;, Bangla814, sans-serif;">Explanation:</strong> An optimal sequence is: press Key 1 three times to get "AAA", then use Key 2 (select all) and Key 3 (copy), followed by Key 4 twice (paste). This results in "AAAAAAAAA", so the total number of A’s is 9.</span></pre>
