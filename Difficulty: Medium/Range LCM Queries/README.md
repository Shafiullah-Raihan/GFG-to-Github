<h2><a href="https://www.geeksforgeeks.org/problems/range-lcm-queries3348/1">Range LCM Queries</a></h2><h3>Difficulty Level : Difficulty: Medium</h3><hr><div class="problems_problem_content__Xm_eO" style="--text-color: var(--problem-text-color);"><p style="font-family: Nunito, Bangla589, sans-serif;"><span style="font-size: 18px; font-family: Nunito, Bangla589, sans-serif;">Given an array <strong style="font-family: &quot;Source Sans 3&quot;, Bangla589, sans-serif;">arr[]</strong>&nbsp; and a list of queries <strong style="font-family: &quot;Source Sans 3&quot;, Bangla589, sans-serif;">queries[][]</strong>. Each query can be one of the following two types:</span></p>
<ul style="font-family: Nunito, Bangla589, sans-serif;">
<li style="font-family: Nunito, Bangla589, sans-serif;"><span style="font-size: 18px; font-family: Nunito, Bangla589, sans-serif;"><strong style="font-family: &quot;Source Sans 3&quot;, Bangla589, sans-serif;">Update Query:</strong> [1, index, value] --&gt; </span><span style="font-size: 18px; font-family: Nunito, Bangla589, sans-serif;">Update the element at position index in the array to the given value.</span></li>
<li style="font-family: Nunito, Bangla589, sans-serif;"><span style="font-size: 18px; font-family: Nunito, Bangla589, sans-serif;"><strong style="font-family: &quot;Source Sans 3&quot;, Bangla589, sans-serif;">Range Query:</strong> [2, L, R] --&gt; </span><span style="font-size: 18px; font-family: Nunito, Bangla589, sans-serif;">Compute and return the <strong style="font-family: &quot;Source Sans 3&quot;, Bangla589, sans-serif;">Least Common Multiple (LCM)</strong> of all elements in the subarray from index <strong style="font-family: &quot;Source Sans 3&quot;, Bangla589, sans-serif;">L</strong> to <strong style="font-family: &quot;Source Sans 3&quot;, Bangla589, sans-serif;">R</strong> (inclusive).</span></li>
</ul>
<p style="font-family: Nunito, Bangla589, sans-serif;"><span style="font-size: 18px; font-family: Nunito, Bangla589, sans-serif;">Process all queries sequentially and return a list containing the results of all Type 2 queries.</span></p>
<p style="font-family: Nunito, Bangla589, sans-serif;"><strong style="font-family: &quot;Source Sans 3&quot;, Bangla589, sans-serif;"><span style="font-size: 18px; font-family: Nunito, Bangla589, sans-serif;">Note:</span></strong><span style="font-size: 18px; font-family: Nunito, Bangla589, sans-serif;">&nbsp;All operations follow 0-based indexing.</span></p>
<p style="font-family: Nunito, Bangla589, sans-serif;"><strong style="font-family: &quot;Source Sans 3&quot;, Bangla589, sans-serif;"><span style="font-size: 18px; font-family: Nunito, Bangla589, sans-serif;">Examples :</span></strong></p>
<pre style="font-family: Nunito, Bangla589, sans-serif;"><strong style="font-family: &quot;Source Sans 3&quot;, Bangla589, sans-serif;"><span style="font-size: 18px; font-family: Nunito, Bangla589, sans-serif;">Input: </span></strong><span style="font-size: 18px; font-family: Nunito, Bangla589, sans-serif;"><span style="font-size: 18px; font-family: Nunito, Bangla589, sans-serif;">arr[] = [2, 3, 4, 6, 8, 16], queries[][] = [[2, 0, 2], [1, 3, 8], [2, 2, 5]]
</span><strong style="font-size: 18px; font-family: &quot;Source Sans 3&quot;, Bangla589, sans-serif;">Output:</strong><span style="font-size: 18px; font-family: Nunito, Bangla589, sans-serif;"> [12, 16]</span><strong style="font-size: 18px; font-family: &quot;Source Sans 3&quot;, Bangla589, sans-serif;">
Explanation: </strong><span style="font-size: 18px; font-family: Nunito, Bangla589, sans-serif;">The queries are processed sequentially, updating the array when required.
</span></span><span style="font-size: 18px; font-family: Nunito, Bangla589, sans-serif;">[2, 0, 2]: LCM of [2, 3, 4] = 12
[1, 3, 8]: array becomes [2, 3, 4, 8, 8, 16]
[2, 2, 5]: LCM of [4, 8, 8, 16] = 16</span></pre>
<pre style="font-family: Nunito, Bangla589, sans-serif;"><strong style="font-family: &quot;Source Sans 3&quot;, Bangla589, sans-serif;"><span style="font-size: 18px; font-family: Nunito, Bangla589, sans-serif;">Input: </span></strong><span style="font-size: 18px; font-family: Nunito, Bangla589, sans-serif;">arr[] = [1, 2, 3, 4],  queries[][] = [[2, 0, 3], [1, 0, 5], [2, 0, 1]]<br style="font-family: Nunito, Bangla589, sans-serif;"></span><span style="font-size: 18px; font-family: Nunito, Bangla589, sans-serif;"><strong style="font-family: &quot;Source Sans 3&quot;, Bangla589, sans-serif;">Output:</strong> [12, 10]<strong style="font-family: &quot;Source Sans 3&quot;, Bangla589, sans-serif;">
Explanation: </strong>The queries are processed sequentially, updating the array when required.
</span><span style="font-size: 18px; font-family: Nunito, Bangla589, sans-serif;">[2, 0, 3]: LCM of [1, 2, 3, 4] = 12
[1, 0, 5]: array becomes [5, 2, 3, 4]
[2, 0, 1]: LCM of [5, 2] = 10<br style="font-family: Nunito, Bangla589, sans-serif;"></span></pre>
