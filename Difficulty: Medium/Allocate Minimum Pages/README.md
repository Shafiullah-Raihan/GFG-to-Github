<h2><a href="https://www.geeksforgeeks.org/problems/allocate-minimum-number-of-pages0937/1">Allocate Minimum Pages</a></h2><h3>Difficulty Level : Difficulty: Medium</h3><hr><div class="problems_problem_content__Xm_eO"><p style="font-family: Nunito, Bangla586, sans-serif;"><span class="cf0" style="font-size: 14pt; font-family: Nunito, Bangla586, sans-serif;">Given an array <strong style="font-family: &quot;Source Sans 3&quot;, Bangla586, sans-serif;">arr[] </strong>of integers, where each element <strong style="font-family: &quot;Source Sans 3&quot;, Bangla586, sans-serif;">arr[i]</strong> represents the number of pages in the i-th book. You also have an integer <strong style="font-family: &quot;Source Sans 3&quot;, Bangla586, sans-serif;">k</strong> representing the number of students. The task is to allocate books to each student such that:</span></p>
<ul style="font-family: Nunito, Bangla586, sans-serif;">
<li style="font-family: Nunito, Bangla586, sans-serif;"><span class="cf0" style="font-size: 14pt; font-family: Nunito, Bangla586, sans-serif;">Each student receives atleast one book.</span></li>
<li style="font-family: Nunito, Bangla586, sans-serif;"><span class="cf0" style="font-size: 14pt; font-family: Nunito, Bangla586, sans-serif;">Each student is assigned a contiguous sequence of books.</span></li>
<li style="font-family: Nunito, Bangla586, sans-serif;"><span class="cf0" style="font-size: 14pt; font-family: Nunito, Bangla586, sans-serif;">No book is assigned to more than one student.</span></li>
</ul>
<p class="pf0" style="font-family: Nunito, Bangla586, sans-serif;"><span class="cf0" style="font-size: 14pt; font-family: Nunito, Bangla586, sans-serif;">The objective is to <strong style="font-family: &quot;Source Sans 3&quot;, Bangla586, sans-serif;">minimize the maximum number of pages </strong>assigned to any student. In other words, out of all possible allocations, find the arrangement where the student who receives the most pages still has the <strong style="font-family: &quot;Source Sans 3&quot;, Bangla586, sans-serif;">smallest possible maximum</strong>.</span></p>
<p class="pf0" style="font-family: Nunito, Bangla586, sans-serif;"><span class="cf0" style="font-size: 14pt; font-family: Nunito, Bangla586, sans-serif;"><strong style="font-family: &quot;Source Sans 3&quot;, Bangla586, sans-serif;">Note:</strong>&nbsp;</span><span style="font-size: 18.6667px; font-family: Nunito, Bangla586, sans-serif;">If it is not possible to allocate books to all students, return <strong style="font-family: &quot;Source Sans 3&quot;, Bangla586, sans-serif;">-1</strong>.</span></p>
<p style="font-family: Nunito, Bangla586, sans-serif;"><span style="font-size: 14pt; font-family: Nunito, Bangla586, sans-serif;"><strong style="font-family: &quot;Source Sans 3&quot;, Bangla586, sans-serif;">Examples:</strong></span></p>
<pre style="font-family: Nunito, Bangla586, sans-serif;"><span style="font-size: 14pt; font-family: Nunito, Bangla586, sans-serif;"><strong style="font-family: &quot;Source Sans 3&quot;, Bangla586, sans-serif;">Input: </strong>arr[] = [12, 34, 67, 90], k = 2
<strong style="font-family: &quot;Source Sans 3&quot;, Bangla586, sans-serif;">Output: </strong>113
<strong style="font-family: &quot;Source Sans 3&quot;, Bangla586, sans-serif;">Explanation: </strong>Allocation can be done in following ways:<br style="font-family: Nunito, Bangla586, sans-serif;">=&gt; [12] and [34, 67, 90] Maximum Pages = 191<br style="font-family: Nunito, Bangla586, sans-serif;">=&gt; [12, 34] and [67, 90] Maximum Pages = 157<br style="font-family: Nunito, Bangla586, sans-serif;">=&gt; [12, 34, 67] and [90] Maximum Pages = 113.<br style="font-family: Nunito, Bangla586, sans-serif;">The third combination has the minimum pages assigned to a student which is 113.</span></pre>
<pre style="font-family: Nunito, Bangla586, sans-serif;"><span style="font-size: 14pt; font-family: Nunito, Bangla586, sans-serif;"><span class="cf0" style="font-family: Nunito, Bangla586, sans-serif;"><strong style="font-family: &quot;Source Sans 3&quot;, Bangla586, sans-serif;">Input:</strong> arr[] = [15, 17, 20], k = 5<br style="font-family: Nunito, Bangla586, sans-serif;"></span><span class="cf0" style="font-family: Nunito, Bangla586, sans-serif;"><strong style="font-family: &quot;Source Sans 3&quot;, Bangla586, sans-serif;">Output: </strong>-1<br style="font-family: Nunito, Bangla586, sans-serif;"></span><span class="cf0" style="font-family: Nunito, Bangla586, sans-serif;"><strong style="font-family: &quot;Source Sans 3&quot;, Bangla586, sans-serif;">Explanation: </strong></span>Since there are more students than total books, it's impossible to allocate a book to each student.</span></pre>
