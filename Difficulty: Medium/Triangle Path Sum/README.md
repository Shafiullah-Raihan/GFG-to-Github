<h2><a href="https://www.geeksforgeeks.org/problems/triangle-path-sum/1">Triangle Path Sum</a></h2><h3>Difficulty Level : Difficulty: Medium</h3><hr><div class="problems_problem_content__Xm_eO"><p style="font-family: Nunito, Bangla1025, sans-serif;"><span style="font-size: 18px; font-family: Nunito, Bangla1025, sans-serif;">Given a&nbsp;<strong style="font-family: &quot;Source Sans 3&quot;, Bangla1025, sans-serif;">triangle</strong>&nbsp;array, return&nbsp;the <strong style="font-family: &quot;Source Sans 3&quot;, Bangla1025, sans-serif;">minimum path sum</strong> to reach from<strong style="font-family: &quot;Source Sans 3&quot;, Bangla1025, sans-serif;"> top </strong>to <strong style="font-family: &quot;Source Sans 3&quot;, Bangla1025, sans-serif;">bottom</strong>.</span></p>
<p style="font-family: Nunito, Bangla1025, sans-serif;"><span style="font-size: 18px; font-family: Nunito, Bangla1025, sans-serif;">For each step, you may move to an adjacent number of the row below. More formally, if you are on index&nbsp;<strong style="font-family: &quot;Source Sans 3&quot;, Bangla1025, sans-serif;"><code style="font-family: Nunito, Bangla1025, sans-serif;">i</code></strong>&nbsp;on the <strong style="font-family: &quot;Source Sans 3&quot;, Bangla1025, sans-serif;">current row</strong>, you may move to either index&nbsp;<strong style="font-family: &quot;Source Sans 3&quot;, Bangla1025, sans-serif;"><code style="font-family: Nunito, Bangla1025, sans-serif;">i</code>&nbsp;</strong>or index&nbsp;<strong style="font-family: &quot;Source Sans 3&quot;, Bangla1025, sans-serif;"><code style="font-family: Nunito, Bangla1025, sans-serif;">i + 1</code></strong>&nbsp;on the <strong style="font-family: &quot;Source Sans 3&quot;, Bangla1025, sans-serif;">next row.</strong></span></p>
<p style="font-family: Nunito, Bangla1025, sans-serif;"><strong style="font-family: &quot;Source Sans 3&quot;, Bangla1025, sans-serif;"><span style="font-size: 18px; font-family: Nunito, Bangla1025, sans-serif;">Examples:</span></strong></p>
<pre style="font-family: Nunito, Bangla1025, sans-serif;"><span style="font-size: 18px; font-family: Nunito, Bangla1025, sans-serif;"><strong style="font-family: &quot;Source Sans 3&quot;, Bangla1025, sans-serif;">Input: </strong>triangle = [[2],[3,4],[6,5,7],[4,1,8,3]]
<strong style="font-family: &quot;Source Sans 3&quot;, Bangla1025, sans-serif;">Output: </strong>11
<strong style="font-family: &quot;Source Sans 3&quot;, Bangla1025, sans-serif;">Explanation:</strong></span>
     <span style="font-size: 18px; font-family: Nunito, Bangla1025, sans-serif;">2
   <u style="font-family: Nunito, Bangla1025, sans-serif;">3</u> 4
  6 <u style="font-family: Nunito, Bangla1025, sans-serif;">5</u> 7
 4 <u style="font-family: Nunito, Bangla1025, sans-serif;">1</u> 8 3
The minimum path sum from top to bottom is 2 + 3 + 5 + 1 = 11.</span></pre>
<pre style="font-family: Nunito, Bangla1025, sans-serif;"><span style="font-size: 18px; font-family: Nunito, Bangla1025, sans-serif;"><strong style="font-family: &quot;Source Sans 3&quot;, Bangla1025, sans-serif;">Input: </strong></span><span style="font-size: 18px; font-family: Nunito, Bangla1025, sans-serif;">triangle = [[10]]
