<h2><a href="https://www.geeksforgeeks.org/problems/make-the-array-beautiful--170647/1">Make the array beautiful</a></h2><h3>Difficulty Level : Difficulty: Easy</h3><hr><div class="problems_problem_content__Xm_eO" style="--text-color: var(--problem-text-color);"><p style="font-family: Nunito, Bangla718, sans-serif;"><span style="font-size: 18px; font-family: Nunito, Bangla718, sans-serif;">Given an array of negative and non-negative integers. You need to make the array beautiful.</span></p>
<ul style="font-family: Nunito, Bangla718, sans-serif;">
<li style="font-family: Nunito, Bangla718, sans-serif;"><span style="font-size: 18px; font-family: Nunito, Bangla718, sans-serif;">An array is beautiful if two adjacent integers, arr[i] and arr[i+1] are either negative or positive. </span><span style="font-size: 18px; font-family: Nunito, Bangla718, sans-serif;">You can do the following operation any number of times until the array becomes beautiful. </span></li>
<li style="font-family: Nunito, Bangla718, sans-serif;"><span style="font-size: 18px; font-family: Nunito, Bangla718, sans-serif;">If two adjacent are different i.e. one of them is negative and other is positive, remove them.&nbsp;</span></li>
<li style="font-family: Nunito, Bangla718, sans-serif;"><span style="font-size: 18px; font-family: Nunito, Bangla718, sans-serif;">Return the beautiful array after performing the above operation.</span></li>
<li style="font-family: Nunito, Bangla718, sans-serif;"><span style="font-size: 18px; font-family: Nunito, Bangla718, sans-serif;">An empty array is also a beautiful array. </span></li>
<li style="font-family: Nunito, Bangla718, sans-serif;"><span style="font-size: 18px; font-family: Nunito, Bangla718, sans-serif;"><span style="font-size: 14pt; font-family: Nunito, Bangla718, sans-serif;">There can be multiple beautiful output arrays. For consistencty with the test cases, scan the array from left to right for removing two adjacent.</span></span></li>
</ul>
<p style="font-family: Nunito, Bangla718, sans-serif;"><span style="font-size: 18px; font-family: Nunito, Bangla718, sans-serif;"><strong style="font-family: &quot;Source Sans 3&quot;, Bangla718, sans-serif;">Examples:</strong></span></p>
<pre style="font-family: Nunito, Bangla718, sans-serif;"><span style="font-size: 18px; font-family: Nunito, Bangla718, sans-serif;"><strong style="font-family: &quot;Source Sans 3&quot;, Bangla718, sans-serif;">Input: </strong>arr[] = [4, 2,-2, 1] <strong style="font-family: &quot;Source Sans 3&quot;, Bangla718, sans-serif;">
Output: </strong>[4, 1]
<strong style="font-family: &quot;Source Sans 3&quot;, Bangla718, sans-serif;">Explanation:</strong> As at indices 1 and 2 , 2 and -2 have different sign, they are removed. And the the final array is: [4, 1].</span>
</pre>
<pre style="font-family: Nunito, Bangla718, sans-serif;"><strong style="font-family: &quot;Source Sans 3&quot;, Bangla718, sans-serif;"><span style="font-size: 18px; font-family: Nunito, Bangla718, sans-serif;">Input: </span></strong><span style="font-size: 18px; font-family: Nunito, Bangla718, sans-serif;">arr[] = [2,-2, -1, 1]</span><strong style="font-family: &quot;Source Sans 3&quot;, Bangla718, sans-serif;"><span style="font-size: 18px; font-family: Nunito, Bangla718, sans-serif;">
Output: </span></strong><span style="font-size: 18px; font-family: Nunito, Bangla718, sans-serif;">[]</span><strong style="font-family: &quot;Source Sans 3&quot;, Bangla718, sans-serif;"><span style="font-size: 18px; font-family: Nunito, Bangla718, sans-serif;">
Explanation: </span></strong><span style="font-size: 18px; font-family: Nunito, Bangla718, sans-serif;">As at indices 0 and 1, 2 and -2 have different sign, so they are removed. Now the array is 1 -1.Now 1 and -1 are also removed as they have different sign. So the final array is empty. </span></pre>
